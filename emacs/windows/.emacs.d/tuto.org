* 1. Leader Key =M-SPC= (hub central)

Aguarde 0.8s apos =M-SPC= para ver o which-key com todas as opcoes.

* 5. Completions (Icomplete vertical, built-in)

No minibuffer, digite palavras-chave separadas por espaco com =completion-styles= = =flex=.

- =C-n= / =C-p= — navegar entre opcoes
- =C-j= — confirmar selecao e sair
- =RET= — confirmar input

* 6. Dicas de Uso Diario

** Git (VC built-in)
=M-SPC g g= abre o VC dir. Comandos uteis:
- =m= — mark file
- =s= — stage (git add)
- =v= — commit (escreve mensagem, =C-c C-c= confirma)
- =P= — push
- =d= — diff
- =l= — log

#+BEGIN_QUOTE
Fluxo tipico: =g g= → =m= (marca tudo) → =s= (stage) → =v= (commit) → =P= (push).
#+END_QUOTE

** Icomplete
=icomplete-vertical-mode= mostra opcoes em lista vertical. =flex= matching permite digitar partes do nome em qualquer ordem.

** VC Gutter
Indicadores de diff no fringe (verde = adicionado, vermelho = removido, amarelo = modificado). Ativo em =prog-mode=.

** HL-Todo
Destaca TODO, FIXME, HACK, NOTE, XXX, BUG em comentarios. Ativo em =prog-mode= e =text-mode=.

** Ace Window
=M-SPC j= mostra letras sobre cada janela — pressione a letra para saltar.

** Tree-sitter
=M-x my/treesit-install-grammars= instala todas as gramaticas configuradas.

** Eshell
=M-SPC e= abre/fecha terminal. Prompt customizado, history persistente, comandos visuais configurados.

** OpenCode
- =M-x opencode= — inicia uma sessao no diretorio do projeto atual.
- =M-x opencode-ask= — cria sessao e envia prompt direto.
- =M-x opencode-ask-contextual= — envia prompt com contexto do buffer.
